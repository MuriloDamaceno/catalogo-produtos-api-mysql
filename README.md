# Catálogo de Produtos API

API REST para gerenciamento de catálogo de produtos, desenvolvida com Node.js, Express e MongoDB.

## Tecnologias Utilizadas

- Node.js
- Express
- MongoDB
- Mongoose
- BCryptjs
- JSON Web Token (JWT)

## Como Rodar o Projeto

### Pré-requisitos
- Node.js instalado
- MongoDB instalado e rodando
